Welcome to my Practice API!

This api provides access to the persons database. It allows you to search by
  • first name
  • last name
  • age
  • or, all results
To use this api in your code you will need a personal API Key. It is very easy to obtain this key. Simply enter a valid email address in the input field and click Submit. Your API Key will arrive shortly!
To use the API without the assistance of .fetch() or an AJAX request visit the test page Here!
To view the docs click Here!
Please keep in mind this API is hosted on Heroku so it could possibly take 30 seconds to wake up when a request is submitted. You can wake up the API Here.
